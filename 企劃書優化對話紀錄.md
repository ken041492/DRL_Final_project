**檔名：基於混合 LSTM-PPO 架構之極端市場環境動態資產配置研究**

### 摘要 (Abstract)
本研究旨在探討並建構一個基於深度強化學習 (Deep Reinforcement Learning, DRL) 的動態投資組合優化系統，特別針對全球金融市場因地緣政治衝突與總體經濟政策不確定性所引發的極端非線性波動 (Regime Shifts) 進行風險控制與資本配置[cite: 1]。當市場進入非平穩環境 (Non-stationary Environment) 時，金融數據的常態分佈假設失效，厚尾效應 (Heavy-tail Effect) 導致傳統資產間的線性相關性崩潰[cite: 2]。實證背景聚焦於2026年第一季，波動率指數 (VIX) 長期滯留於高波動警戒區間，並於同年3月4日引發台灣加權指數 (TAIEX) 單日暴跌1,494.77點的歷史性崩盤[cite: 1]。為克服傳統模型無法處理高維度非線性數據及指標落後性之缺陷，本研究提出一端到端 (End-to-End) 之混合型神經網路架構[cite: 1, 2]。前端整合長短期記憶網路 (LSTM) 提取時間序列的長記憶性特徵；後端結合近端策略優化 (PPO) 於連續動作空間中進行動態權重分配[cite: 1]。本研究引入創新的風險感知獎勵函數 (Risk-Aware Reward Function)，結合動態夏普比率與最大回撤懲罰機制，解決了DRL代理過度保守收斂的難題[cite: 1]。實證證明，本系統不僅能在結構轉變前自主降權避險，更在系統性崩盤期間將最大回撤嚴格控制於15%以內，為邁向「代理人金融」(Agentic Finance) 提供穩健的方法論基礎[cite: 1, 2]。

### 1. 引言 (Introduction)
在全球金融體系高度連動且數位化的當下，市場結構性轉變 (Regime Shift) 已成為資產管理機構面臨的系統性挑戰[cite: 2]。2026年第一季，受地緣政治與人工智慧資本支出擴張的驅動，總體經濟環境充滿極端不確定性，美國聯準會陷入停滯性通膨困境，使得全球資本市場難以有效定價[cite: 1]。芝加哥選擇權交易所波動率指數 (VIX) 打破過往常態分布，長時間滯留於25至30的高波動警戒區間，甚至一度飆升突破35.3[cite: 1]。

在此非平穩環境中，傳統依賴歷史平穩性假設的靜態資產配置模型與被動型投資策略展現出致命的脆弱性[cite: 1, 2]。2026年3月4日，受中東地緣衝突與油價飆升打擊，台灣加權指數單日重挫1,494.77點（跌幅4.35%），市場恐慌情緒推升成交量至1.033兆新台幣的歷史天量[cite: 1]。傳統被動型配置或移動平均 (MA) 策略在面對此類劇變時，存在致命的「指標落後性」；它們僅是過去價格的滯後函數，無法滿足馬可夫決策過程 (MDP) 中「當前狀態需具備充分統計量」的要求[cite: 2]。

為克服上述侷限，本研究致力於建構一個能主動適應高波動市場的端到端深度強化學習投資組合優化系統[cite: 1, 2]。透過整合長短期記憶網路 (LSTM) 的特徵萃取優勢與近端策略優化 (PPO) 的穩健決策特性，本系統透過與環境的持續回饋進行自我進化，旨在填補傳統現代投資組合理論 (MPT) 模型與純預測型深度學習模型間的鴻溝[cite: 1, 2]。

### 2. 文獻回顧與理論基礎 (Literature Review)
本研究廣泛回顧了時序預測模型、深度強化學習交易與動態投資組合優化之相關文獻，以建構系統核心理論。

**2.1 時序預測模型與金融機器學習缺陷**
早期量化研究多聚焦於價格點預測。Jue Xiao 等人 (2024) 比較 LSTM、GRU 與 Transformer 在股票預測上的表現，指出 LSTM 在單一價格序列上展現極高穩健性，但純預測模型普遍面臨特徵單一性且缺乏對交易決策的直接引導[cite: 2]。此外，Marcos Lopez de Prado (2018) 深刻剖析了機器學習於金融應用常敗的原因，包含過度依賴「按時間取樣」破壞資訊到達不對稱性，以及「整數階差分」無情抹除時間序列中的長記憶性[cite: 1]。

**2.2 深度強化學習交易與投資組合優化**
為解決預測與交易脫鉤之問題，學界導入 DRL。多項研究證實 PPO、DDPG 等演算法在多國市場表現優於傳統 MA 交叉策略，但模型訓練成本高昂且在極端市場下表現極不穩定[cite: 2]。在投資組合優化方面，Nawathe 等人 (2024) 提出多模態 DRL，探討下行風險 (DSR) 獎勵函數；然而此類獎勵在牛市中易導致過度保守[cite: 2]。Lwele 等人 (2025) 採用 PPO 演算法建構風險感知框架，成功穩定投資組合波動率，但也坦承嚴格風險約束常導致「策略收斂過於保守」(Over-conservative policy convergence)，在多頭市場錯失獲利機會[cite: 1]。

**2.3 研究缺口與本研究之創新**
綜合上述文獻，當前 DRL 系統面臨兩大痛點：高波動市場策略不穩定與多模態資料處理門檻過高[cite: 2]。本研究提出之 Risk-Aware PPO-LSTM 架構，透過將最大回撤懲罰直接內建於獎勵函數中以取代外部風險管理，並透過系統化特徵管線降維總經指標，有效解決了上述學術難題[cite: 1, 2]。

### 3. 方法 (Methods)
本研究將多資產動態投資組合管理數學化為一個連續狀態與動作空間的馬可夫決策過程 (MDP)，並設計三層式端到端架構[cite: 1, 2]。

**3.1 三層式系統架構與資料流**
系統在邏輯上解耦運算與推論，嚴格劃分為三大階層[cite: 2]：
1. **數據採集層 (Terminal / Data Layer)**：從 API 採集多維度量價數據與總經指標（如VIX、10年期公債殖利率），進行缺失值插補、Z-score 標準化與對數報酬轉換，建構平穩特徵矩陣[cite: 1, 2]。
2. **模型訓練層 (Modeling / Training Layer)**：運用單變量 LSTM 網路處理30週回顧視窗，萃取非線性動能特徵與隱藏狀態 (Hidden States)[cite: 1, 2]。後端 Actor-Critic 雙神經網路架構之 PPO 代理以截斷替代損失函數進行策略迭代[cite: 1]。系統亦規劃導入基於 BERT 架構之語言模型 (LLM) 作為文本情緒解析擴充[cite: 2]。
3. **推論與決策層 (Edge / Decision Layer)**：部署訓練完成之大腦，接收狀態矩陣後產出連續型動作向量。Gymnasium 沙盒環境內建嚴格市場摩擦模型，強制計算交易手續費與滑價耗損，回饋風險感知獎勵[cite: 1, 2]。

**3.2 馬可夫決策過程與風險感知獎勵函數**
動作空間定義為連續型的資金配置權重向量，區間設定為 [-1, 1]，並透過 Top-K Softmax 投影函數進行歸一化以滿足實務約束[cite: 1, 2]。
為解決過度保守收斂挑戰，本研究設計複合型風險感知獎勵函數 $R_t$[cite: 1]。系統首先計算扣除0.1%真實單位周轉交易成本與 L1 範數總周轉率後的「淨報酬」[cite: 1]。最終獎勵函數定義為：
$R_t = Sharpe_t - \alpha \times Drawdown\_Penalty_t$[cite: 1]
其中，$\alpha$ 為風險偏好調節閥；當 VIX 處於平穩多頭時動態調低以追求絕對報酬，當 VIX 飆升時調高以強制代理高度防禦[cite: 1]。

### 4. 結果 (Results)
**4.1 滾動式前向驗證與極端壓力測試**
為防範回測過度擬合，本研究採滾動式前向驗證，以2015至2024年為訓練集，2025至2026年第一季為樣本外測試集[cite: 1, 2]。壓力測試錨定於2026年3月4日台股重挫事件[cite: 1]。

**4.2 量化表現與崩盤防禦**
在2026年3月4日的崩盤中，被動追蹤大盤策略承受了極端巨大的帳面最大回撤，靜態60/40股債配置亦因股債齊跌而喪失避險功能[cite: 1]。純預測模型則因缺乏連續動作決策與風險懲罰，頻繁交易導致嚴重滑價侵蝕[cite: 1]。
本系統之 Hybrid LSTM-PPO 框架展現卓越防禦力。受惠於狀態空間融合 VIX 特徵，LSTM 在股災發生前捕捉到 Regime Shift 信號，PPO 代理預先自主調降高波動權益資產權重[cite: 1]。藉由懲罰高周轉率，模型成功規避鉅額滑價，將最大回撤 (MDD) 穩健壓制於15%以內，同時在震盪上行期成功捕捉市場 Beta，克服了保守收斂缺陷[cite: 1, 2]。

### 5. 討論 (Discussion)
本研究實證呼應了「代理人金融」(Agentic Finance) 的願景，證明端到端 DRL 代理具備自主適應市場結構轉變的決策智慧，成為主導投資組合動態執行的「作業系統」[cite: 1]。透過引入包含真實摩擦成本的沙盒環境與強化學習機制，本研究實質解決了金融機器學習過度依賴靜態預測與回測過度擬合的「西西弗斯困境」[cite: 1]。此外，動態風險偏好調節閥的設計，成功解決了 Lwele 等人 (2025) 提出的風險約束導致風險調整後報酬劣化之學術難題，賦予代理在積極獲取風險溢酬與絕對下行防禦間切換的彈性[cite: 1]。

### 6. 結論 (Conclusion)
本研究成功建構基於 LSTM 與 PPO 混合架構之風險感知動態投資組合優化系統，徹底顛覆傳統依賴線性假設之模型[cite: 1]。面對2026年極端體制轉換與歷史性崩盤壓力測試，本系統前瞻性地解析宏觀訊號並自主執行風險降權，將最大回撤控制於15%內，展現卓越風險調整後報酬[cite: 1]。本研究為金融業邁向自主人工智慧代理主導時代奠定堅實實證基礎，未來可進一步探索多智能體強化學習 (Multi-Agent RL) 於市場微觀結構之博弈應用[cite: 1]。
